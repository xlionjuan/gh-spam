# GitHub Bulk Spam Abuse Report

## Summary

This is a follow-up report for additional spam repositories identified as part of the same coordinated gambling spam attack network documented in previous reports. These repositories remain active and contain gambling/adult content spam through GitHub Issues.

**Estimated Total Spam Issues:** 50,000+ across 216 repositories

---

## Attack Pattern Description

1. **Spam Type:** Coordinated gambling/adult content spam via GitHub Issues
2. **Keywords Observed:**
   - `彩民`, `彩票`, `主管开户`, `代理` (gambling platform promotions)
   - `AG真人`, `百家乐`, `炸金花`, `澳门赌场` (casino/gambling terms)
   - `皇冠信用盘`, `pg电子`, `PG电子` (credit card/gambling platform)
   - `澳洲幸运10`, `加拿大28`, `极速赛车`, `快3` (lottery/racing gambling)
   - `91y银商`, `集结号`, `芒果棋牌` (gaming platform merchants)
   - `计划导师`, `回血上岸`, `包赔` (gambling scheme promotions)
   - `宝塔面板`, `售后服務` (fake tech/home appliance spam as cover)
3. **Method:** Automated bot accounts creating issues with gambling promotion content
4. **Naming Pattern:** Random-looking usernames and repo names
