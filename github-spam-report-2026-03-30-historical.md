# GitHub Bulk Spam Abuse Report - Historical Spam

## Summary

This is a follow-up report to my previous submission, covering additional spam repositories that were identified as part of the same coordinated gambling spam attack network but belong to earlier time periods (2023-2025).

**Estimated Total Historical Spam Issues:** ~12,000+ issues across 15+ repositories

---

## Attack Pattern Description

Same coordinated attack pattern as the primary report:
1. **Spam Type:** Coordinated gambling/adult content spam via GitHub Issues
2. **Keywords Observed:** 
   - `彩民`, `彩票`, `彩票导师` (gambling platform promotions)
   - `AG真人`, `百家乐` (casino/gambling terms)
   - Domain typosquatting (e.g., `baidu.com`, `baidu`)
3. **Method:** Automated bot accounts creating issues with gambling promotion content
4. **Naming Pattern:** Random-looking usernames and repo names
